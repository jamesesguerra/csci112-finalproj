# csci 112 final proj

## setup

### clone the repo
```sh
# ssh
git clone git@github.com:jamesesguerra/csci112-finalproj.git

# https
git clone https://github.com/jamesesguerra/csci112-finalproj.git
```

### cd into folder
```sh
cd csci112-finalproj
```

### install dependencies
```sh
pip install -r requirements.txt
```
